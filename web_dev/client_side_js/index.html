How to Make an Oboe Reed

  1. Gouge the cane.

    2. This is a gouger.

    This is what tube cane looks like, or cane before it is gouged.

    This is gouged cane.

  2. Shape the cane.

    3. Shape the cane using a shaper tip, handle and razor blade.

    Shaped oboe cane.

    Oboe shaper tip. I recommend using a Mack or Mack Pfeiffer shaper tip.

  3. Tie the cane to a staple.
  4. Clip the tip and start scraping.
    • Using a razor blade, cut the tip open. The total length of the reed should be no more than 73 mm.
    • With the razor blade, cut the ears off of the cane so that each side is flat, smooth and straight.
    • Begin by scraping all of the bark, or shiny hard surface, off the reed.
  5. Scrape out the windows, leaving the rails and the backbone. Also, scrape the tip, scraping towards the corners at a 45 degree angle.
